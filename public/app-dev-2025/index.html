App Development Competition - 2025

Showcase your skills and compete with the best app developers

Development Rules

No AI Tools

  • Do not use AI tools such as ChatGPT, GitHub Copilot, Gemini, Notion AI, etc. for code generation.
  • Your app must be 100% created by you, including code and content.
  • If any part is found to be AI-generated, your submission will be disqualified.

No App Builders

  • Drag-and-drop app builders like Thunkable, Appy Pie, or similar platforms are not allowed.
  • Use only code-based development.

Allowed Technologies

  • Native: Kotlin/Java (Android), Swift (iOS)
  • Cross-platform: Flutter, React Native, Xamarin, etc.
  • Backend technologies if required (Firebase, Node.js, etc.)

Original Work Only

  • The entire project must be your original work.
  • Do not copy from templates, tutorials, or GitHub repositories.
  • Plagiarism will result in immediate disqualification.

Platform Requirements

  • Apps can be developed for Android, iOS, or cross-platform.
  • Must include APK file (Android) or simulator demo (iOS).
  • Include all source files in your submission.

Judging Criteria

Innovation & Creativity

Originality of concept, unique features, and creative solutions to problems.

Functionality & Performance

How well the app works, smoothness of operation, and absence of critical bugs.

User Experience

Intuitive design, ease of use, and overall user satisfaction.

Code Quality

Clean, efficient, and well-structured code following best practices.

Submission Guidelines

Submission Requirements

  • Submit a public GitHub repository with your complete app code. It must match what's shown in your video.
  • Submit an MP4 or MOV video demonstrating your app's functionality. Use your voice to explain all features to prove it's your original work.
  • Include either an APK file (Android) or Xcode project (iOS) or equivalent for your platform.

Deadline

  • Submit your project on or before 30 September 2025.
  • Late entries will not be accepted.
  • The project submission form will automatically send a confirmation to your email.
