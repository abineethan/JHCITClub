Web Developing Competition - 2025

Showcase your skills and compete with the best web developers

Development Rules

No AI Tools

  • Do not use AI tools such as ChatGPT, GitHub Copilot, Gemini, Notion AI, etc.
  • Your website must be 100% created by you, including code and content.
  • If any part is found to be AI-generated, your submission will be disqualified.

No Web Builders

  • Canva, Wix, WordPress, Google Sites, and other drag-and-drop website builders are not allowed.
  • Use only code-based development.

Allowed Technologies

  • HTML, CSS, JavaScript
  • Optional: Frameworks like Bootstrap, Tailwind, React, etc.
  • Backend technologies if required (Node.js, PHP, etc.)

Original Work Only

  • The entire project must be your original work.
  • Do not copy from templates, other websites, or GitHub repositories.
  • Plagiarism will result in immediate disqualification.

Hosting

  • You may host your website online (e.g., using GitHub Pages or Netlify).
  • Hosting is not required but recommended for better presentation.
  • Include all source files in your submission.

Judging Criteria

Creativity & Visual Design

Originality, aesthetic appeal, and innovative design solutions will be evaluated.

Functionality & Usability

How well the website works, user experience, and interactive elements.

Code Quality

Clean, efficient, and well-structured code following best practices.

Submission Guidelines

Submission Requirements

  • Submit a public GitHub repository with your complete website code. It must match what's shown in your video for judging.
  • Submit an MP4 or MOV video showing your website. Use your voice to explain all features and functionality to prove it's your original work.

Deadline

  • Submit your project on or before 31 July 2025.
  • Late entries will not be accepted.
  • The project submission form will automatically send a confirmation to your email.
